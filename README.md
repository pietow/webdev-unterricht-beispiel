# Webdev Unterricht – ausgewählte Unterrichtseinheiten verschiedener Kurse

Lehr- und Beispielmaterial für einen praxisorientierten Webentwicklungs-Kurs.  
Die Inhalte sind Lehrthemen aus drei verschieden Kursen, die ich unterrichtet habe.
Ausgewählte live Code Beispiele sind als git submodule enthalten.

## Kursmodule

| Modul | Thema | Inhalt |
|---|---|---|
| `1-CSS` | Styling & Layout | CSS-Grundlagen, Flexbox, Grid |
| `2-NEXTJS` | Frontend-Entwicklung | Next.js, React-Komponenten, App Router, TypeScript |
| `3-CLOUD` | Deployment & Infrastruktur | Docker, Shell-Scripting, Cloud-Deployment, CI/CD |


## Technologien

- **Frontend:** HTML, CSS, Next.js, React, TypeScript, Tailwind CSS
- **Infrastruktur:** Docker, Shell-Scripting, Terraform
- **Lehrformat:** Jupyter Notebooks und Markdown für interaktive Code-Beispiele

## Verwendung
```bash
# Submodule nach dem Klonen initialisieren
git clone --recurse-submodules https://github.com/pietow/webdev-unterricht-beispiel
```

## Ziel

Dieses Repository vermittelt modernen Webentwicklungs-Stack von den CSS-Grundlagen  
bis zum Cloud-Deployment – praxisnah, strukturiert und mit echten Beispielprojekten.
